# manyfold 0.1.36

manyfold 0.1.36 is a compatibility patch for 0.1.35.

## Fixes

- Restores the legacy top-level stream node exports used by downstream
  runtimes, including `EmptyNode`, `ConstantNode`, `MergeNode`, `Timer`, and
  `StreamNode`.
- Keeps `Timer` as a graph-facing stream helper without shadowing the internal
  thread timer used by coalesced stream scheduling.
- Restores the `EventStream` lock-backed subscriber lifecycle so callback
  streams remain bounded and deterministic under concurrent subscribe/dispose.

The native graph retention fix from 0.1.35 remains unchanged: `latest_only` and
non-replayable routes bound native history, payload retention, and route audit
state for long-running sensor streams.
