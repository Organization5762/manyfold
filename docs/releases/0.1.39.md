# manyfold 0.1.39

manyfold 0.1.39 restores Heart runtime compatibility and removes a hot-path
payload-retention sweep that dominated high-rate sensor workloads.

## Fixes

- Adds `pairwise()` to `FluentStream`, matching the legacy timer stream API used
  by gamepad d-pad navigation.
- Keeps payload retention enforcement route-local instead of scanning the full
  payload registry on every write.
- Allows typed decoding to pass through already-decoded pipeline values, so
  route-to-route command wiring can publish without wrapping non-envelope values
  as `TypedEnvelope`.

- `uv run ruff check python/manyfold/graph.py tests/test_graph_reactive.py`
