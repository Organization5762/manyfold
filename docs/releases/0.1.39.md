# manyfold 0.1.39

manyfold 0.1.39 adds the first Rust-backed architecture PubSub stream surface.

## Features

- Adds `manyfold.architecture.PubSub`, a stream handle backed by native PubSub
  delivery, SQL stream state, and row-oriented query helpers.
- Supports ephemeral UUID5 topics, lazy schema inference from the first
  structured model publish, and SQL-backed `latest()` rows with attribute and
  item access such as `latest.degrees` and `latest["degrees"]`.
- Materializes structured model fields into SQL columns while retaining the
  encoded payload bytes, so callers can use helpers such as
  `stream.average(field="degrees")` or write direct SQL against `stream`.

## Notes

- The current transport encoding is FlatBuffers under the hood. The public API
  keeps the logical model and row surface separate so the stream processor can
  choose a different physical format later.
