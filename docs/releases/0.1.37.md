# manyfold 0.1.37

manyfold 0.1.37 is a compatibility patch for downstream Heart runtimes.

## Fixes

- Restores the top-level `BehaviorSubject`, `drain_frame_thread_queue`, and
  `shutdown` exports that Heart imports directly.
- Keeps the 0.1.35 native graph retention fix and the 0.1.36 stream-node
  compatibility restore unchanged.
