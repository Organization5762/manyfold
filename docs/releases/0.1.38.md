# manyfold 0.1.38

manyfold 0.1.38 completes the Heart compatibility repair for fluent timer
streams.

## Fixes

- Restores fluent stream operators used by Heart navigation and renderer
  providers, including `scan`, `start_with`, `do_action`, `take`, and
  `switch_latest`.
- Keeps timer disposal daemon-safe so short-lived fluent timers do not keep test
  or utility processes alive.
